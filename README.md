# Instalação

Para instalar localmente:

```
dotnet new -i ./content/
```

Para instalar utilizando o nuget.org:

```
dotnet new -i PlusUltra.Microservice
```

E então para usar utilizar:

```
dotnet new plus-ms -o ProjetoName
```
